# Projects-for-AI-in-Healthcare

This repository contains a collection of **applied AI and deep learning projects in healthcare**, with a focus on **medical image analysis** for disease classification and segmentation. The projects demonstrate end-to-end workflows, including data preprocessing, model development, training, evaluation, and experimentation using modern deep learning techniques.
